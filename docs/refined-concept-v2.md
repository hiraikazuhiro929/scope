# Work Diary v2: 匿名作業日記アプリ（改良版）

## あなたの追加要望

```yaml
1. 名前は決められる（でも匿名感）
   → ニックネーム自由、本名不要

2. 写真1枚添付可能
   → 作業風景、デスク、成果物など

3. 強い言葉を自動で柔らかく変換
   → ポジティブな空間を保つ
```

## コンセプト（更新版）

### アプリ名: 「Work Diary」（仮）

```yaml
コンセプト:
  「匿名で作業を記録、写真で共有、いいねで応援し合う優しい空間」
```

## 機能仕様（2-3日版）

### 核心機能

```yaml
1. 今日の記録を投稿
   - テキスト（200文字まで）
   - 写真1枚（任意）
   - カテゴリ選択（任意）

2. 他人の記録をランダムで見る
   - スワイプで次へ
   - いいねボタン

3. マイページ
   - カレンダーで振り返り
   - 連続記録日数
   - 累計いいね数

4. プロフィール
   - ニックネーム（自由に設定）
   - アイコン（選択式または写真）
   - 一言紹介（50文字）

5. 言葉の自動変換
   - 強い言葉 → 柔らかい言葉
   - ネガティブ → ポジティブ
```

## 言葉の自動変換機能

### 目的

```yaml
問題:
  SNSは荒れやすい
  - 強い言葉が飛び交う
  - ネガティブが伝染する
  - 疲れる空間になる

解決:
  ✅ 自動で優しい言葉に変換
  ✅ ポジティブな空間を保つ
  ✅ みんなが安心して投稿できる
```

### 変換例

```yaml
強い言葉 → 柔らかい言葉:

「クソ難しかった」
→ 「とても難しかった」

「マジでつらい」
→ 「少し大変だった」

「全然できない」
→ 「まだ上手くいかない」

「バカみたい」
→ 「ちょっと失敗した」

「死ぬほど疲れた」
→ 「とても疲れた」

「最悪だ」
→ 「大変だった」

ネガティブ → ポジティブ変換（オプション）:

「できなかった」
→ 「挑戦した」

「失敗した」
→ 「学んだ」

「諦めた」
→ 「一旦休憩」
```

### 実装方法

#### Phase 1: シンプル版（2-3日で実装）

```javascript
// 強い言葉のフィルタリング（正規表現）
const softWords = {
  'クソ': '',
  'マジで': 'とても',
  '死ぬほど': 'とても',
  'バカ': 'ちょっと',
  '最悪': '大変',
  'つらい': '大変',
  'きつい': '大変',
  'ムカつく': 'イライラする',
  'うざい': '気になる',
  // ... 50-100個の辞書
};

function softifyText(text) {
  let result = text;
  for (const [harsh, soft] of Object.entries(softWords)) {
    result = result.replace(new RegExp(harsh, 'g'), soft);
  }
  return result;
}

// 使い方
const original = "クソ難しかったけど頑張った";
const softened = softifyText(original);
// → "難しかったけど頑張った"
```

#### Phase 2: AI版（後で検討）

```yaml
使用:

コスト:
  1投稿あたり: $0.0001
  100投稿/日: $0.01/日 = $0.3/月

メリット:
  ✅ 自然な言い換え
  ✅ 文脈を理解

デメリット:
  ❌ コストかかる
  ❌ APIレイテンシ
  ❌ Phase 1では不要
```
