# Scope - UI設計案

このドキュメントには、ホーム画面のUI設計案を複数記載しています。

---

## 案1: バブルUI（現在の案）

### コンセプト
- ふわふわ浮かぶ丸いサムネイル
- 「覗き見」感を視覚的に表現
- 楽しい、ワクワクする体験

### レイアウト
```
┌─────────────────────────┐
│ 🔍 Scope  [カテゴリ▼]   │ ← ヘッダー
├─────────────────────────┤
│                         │
│    ◯ サムネイル丸       │
│  ◯        ◯            │  各丸をタップ→
│        ◯                │  フルスクリーン
│  ◯           ◯          │  動画再生
│         ◯               │
│    ◯          ◯         │  ふわふわ浮かぶ
│               ◯         │  アニメーション
│  ◯                      │
│                   [🔄]  │ ← リフレッシュボタン
└─────────────────────────┘
```

### 特徴
- ✅ 覗き見感が強い
- ✅ 独自性がある
- ✅ 楽しいアニメーション
- ❌ SNS感が弱い
- ❌ 見た目がごちゃごちゃしがち
- ❌ 実用性やや低い

```
│                         │
```

```
┌─────────────────────────────────┐
```

- ✅ SNS感が強い（Instagram, BeRealみたい）
- ✅ 見慣れたUI、使いやすい
- ✅ ちるい、ほんわか
- ✅ スクロールで一覧性が高い
- ❌ 独自性が少ない
- ⚠️ 覗き見感はやや弱い

### 参考アプリ
- Instagram Reels（発見タブ）
- BeReal（友達の投稿一覧）
- TikTok（プロフィール画面）

## 案3: TikTok/Shorts風フルスクリーン（即再生）

### コンセプト
- アプリを開いたら即動画再生
- 縦スワイプで次々見る
- 最もシンプル

### レイアウト
```
┌─────────────────────────┐
│ [カテゴリ: すべて ▼]    │ ← 上部
│                         │
│                         │  ← 右側
│   フルスクリーン動画      │  [👍 12]
│   （自動再生）           │  [⋯]
│                         │
│                         │
│ [@nickname]             │ ← 下部
│ [「ロゴデザイン中...」]   │
└─────────────────────────┘
↑↓ スワイプで動画切り替え
```

### 特徴
- ✅ 最もシンプル
- ✅ TikTok/Shortsと同じUX
- ✅ 動画に集中できる
- ❌ 「選ぶ楽しさ」がない
- ❌ データ通信量が多い
- ❌ 即再生は唐突かも

## 案4: 横スクロールカルーセル

### コンセプト
- メインは1つの大きなカード
- 左右にスワイプで次の動画へ
- 下に小さいサムネイル一覧

### レイアウト
```
┌─────────────────────────┐
│ Scope    [カテゴリ▼]     │
├─────────────────────────┤
│                         │
│   ┌─────────────┐       │
│   │             │       │ ← メインカード
│   │  [サムネイル] │       │   （大きめ）
│   │             │       │
│   │  @username  │       │
│   │  👍 25      │       │
│   └─────────────┘       │
│                         │
│ ◯ ◯ ◯ ● ◯ ◯ ◯         │ ← 下部に小さいサムネイル
│                         │   （横スクロール）
└─────────────────────────┘
← → 左右スワイプ
```

### 特徴
- ✅ 1つの動画に集中できる
- ✅ 見やすい
- ⚠️ 左右スワイプは珍しい
- ❌ 一覧性が低い

## 推奨案: 案2（グリッドUI）

### 理由
1. **SNS感が強い**: Instagram, BeRealなど人気SNSの使い慣れたUI
2. **ちるい・ほんわか**: 白背景、余白、柔らかいデザイン
3. **使いやすい**: スクロールで一覧、タップで再生
4. **覗き見感**: サムネイル一覧から「どれ見よう？」と選ぶ楽しさ
5. **データ節約**: タップするまで動画読み込まない
6. **開発しやすい**: FlashListで簡単に実装可能

### タップ後の動画再生
案2でも、サムネイルをタップすると**TikTok/Shorts風のフルスクリーンモーダル**が開き、縦スワイプで次の動画へ移動できます。

```
いいねアイコン: #E74C3C（レッド）
```

## フォント

- ヘッダー: 太字、18-20pt
- ユーザー名: 中太字、14pt
- いいね数: 通常、12pt
- 全体: 丸ゴシック系（優しい印象）
